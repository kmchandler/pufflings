## Local Development
Local development for android should be easy. You can use an emulator OR use your device. 

- Step 1
Ensure that your device is in development mode and USB tethering is enabled.
- Step 2
Plug in the device and make sure that it can be shared with your computer. 
- Step 3
Open Chrome and navigate to chrome://inspect#devices
Click on Port forwarding...
In the box labeled Port add 3000.
In the box labeled ip address and port add localhost:3000.
This will forward all localhost 3000 traffic from your connected phone to the localhost server.

You should now be able to run this app and it will install and run the application on your phone.
Also be sure to have the backend runnin as well.
